# Cliente web Futbol Predice BI

Aplicacion React con Vite, rutas protegidas, tablero, predicciones, equipos, eventos y gestion de usuarios.

- Puerto: `5173`.
- API del navegador: `/api`.
- Proxy de desarrollo: `/api` hacia `http://localhost:3001`.
- Inicio: `npm.cmd run desarrollo:cliente` desde la raiz.

Despues de modificar `cliente-web/.env`, reinicia Vite.
